# سورس کد سایت

یک اپلیکیشن Flutter برای دریافت، نمایش، کپی، ذخیره و اشتراک گذاری سورس HTML یک صفحه وب.

## قابلیت ها

- دریافت سورس HTML از لینک وارد شده
- نمایش سورس در محیط شبیه کد ادیتور
- کپی کامل سورس
- ذخیره خروجی به صورت فایل HTML در اندروید
- اشتراک گذاری سورس
- پشتیبانی از حالت روشن و تاریک سیستم
- رابط راست به چپ و فارسی

## پیش نیاز توسعه

- Flutter 3.44.1 یا نسخه سازگار جدیدتر
- Dart SDK سازگار با Flutter
- Android SDK
- Java 17

## اجرای پروژه روی سیستم خودت

```bash
flutter pub get
flutter run
```

## ساخت APK روی سیستم خودت

```bash
flutter build apk --release --split-per-abi
```

بعد از build، فایل های APK در این مسیر ساخته میشوند:

```text
build/app/outputs/flutter-apk/
```

## انتشار خودکار APK در GitHub Release

این پروژه workflow آماده دارد:

```text
.github/workflows/android-apk-release.yml
```

با ساخت tag مثل `v1.0.0`، GitHub Actions پروژه را build میکند و فایل های APK را داخل Release همان نسخه قرار میدهد.

## نسخه های APK

وقتی خروجی split-per-ABI ساخته شود، معمولا سه APK داری:

- `arm64-v8a`: مناسب بیشتر گوشی های اندرویدی جدید
- `armeabi-v7a`: مناسب بعضی گوشی های قدیمی تر
- `x86_64`: مناسب شبیه سازها و بعضی دستگاه های خاص
